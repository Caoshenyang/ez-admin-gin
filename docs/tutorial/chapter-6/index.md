# 第 6 章：核心系统模块

前面已经把平台基础、认证、接口权限、组织体系和数据权限的骨架打好了。现在开始把这些能力落到真正的系统模块里。

::: tip 本章怎么验证
本章完成后，用户、角色、菜单、配置、文件、日志、公告这些系统模块应该都能在最终结构下继续工作，并为后续前端页面和业务模块接入提供稳定能力。
:::

## 本章会解决什么

- 用户模块如何接入部门、岗位和数据权限
- 角色模块如何管理数据范围
- 菜单和按钮权限如何与接口权限保持一致
- 配置、文件、日志、公告这些系统能力如何继续复用

## 本章下一步

系统模块完成后，教程会进入 [第 7 章：前端企业级管理台](../chapter-7/)。
