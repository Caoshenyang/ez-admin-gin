# 从零搭建教程

这条主线会从空仓库开始，一步步搭出 `EZ Admin Gin`：一个面向个人项目快速上线的通用后台管理系统底座。

::: warning 验证优先
每一节都应该能被手动验证。只写“执行命令”不够，还要写清楚执行后应该看到什么。
:::

## 怎么推进

当前阶段先完成 `tutorial/` 主线。等主线全部跑通后，再回头整理“开始这里”、参考手册和路线图。
